初めての投稿

Github Pagesで初めての投稿をしてみました！ これまで自分の研究だったりインターンだったりでまとめて時間を取れていなかったけど、修論がひと段落したので、自分のポートフォリオとして作ってみました。

早速書いてみる

ついでにこのページの作り方を説明していきたいと思っています。 このページはhugoという静的サイトを簡単に作ってくれるものです。Goというプログラミン言語で書かれていてとてもサイトの生成が早いらしい。 静的サイトジェネレータと言われています。

hugoのインストール

私はmacで作成したので、macでの作成方法になります。

brew install hugo

でインストールできます。簡単。
